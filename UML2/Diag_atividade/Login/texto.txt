Nesse diagrama, o usuário coloca seu login, logo após o login é validado, se der erro, aparece uma mensagem para o usuário recuperar a senha, caso der certo o usuário é redirecionado para a tela de recuperação, se não aparece uma mensagem para ele fazer o cadastro, se ele clicar nessa mensagem ele é redirecionado para a tela de cadatro. Caso o login esteja correto, o usuário é redirecionado para a tela principal do site/app.
